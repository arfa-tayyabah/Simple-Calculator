### Simple-Calculator
A basic  calculator that performs arithmetic operations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Usage


Follow the prompts to enter:
1. First number
2. Operation (+, -, *, /)
3. Second number

Example:

Enter first number: 10
Enter operation: +
Enter second number: 5
Result: 15.0
